# Laptop-Price-Prediction-Data-Science-Machine-Learning-in-R
This project focuses on predicting laptop prices based on real-world specifications using R. It demonstrates a complete data science workflow: data cleaning, exploratory data analysis (EDA), feature engineering, and predictive modeling.

Key highlights include:

Data Preprocessing: Handling missing values, converting categorical variables, and standardizing numeric features.
Exploratory Analysis & Visualization: Price distributions, brand comparisons, RAM/CPU influences, and correlation heatmaps.
Machine Learning Models: Linear Regression, Random Forest, and Gradient Boosting for regression tasks.
Evaluation Metrics: RMSE, MAE, and R² for robust model assessment.
Tools & Libraries: tidyverse, tidymodels, randomForest, xgboost, ggplot2, corrplot, janitor, skimr.

Objective:
The project aims to provide accurate laptop price predictions while uncovering the most significant features affecting pricing, making it suitable for data-driven insights, portfolio demonstration, or ML learning purposes.

Dataset:
Includes key features such as Company, Product, TypeName, Inches, Ram, OS, Weight, Price_euros, CPU_company, CPU_freq, PrimaryStorage, SecondaryStorage, GPU_company, GPU_model, and more.
